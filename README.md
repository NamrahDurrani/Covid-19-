# Covid-19-
Covid-19 project on  country-wise cases
Predicts new COVID-19 cases for each country (Regression)
   - Model used: Random Forest Regressor
   - Handles correlated features and evaluates using R² and MSE

## Features
- Data cleaning and preprocessing
- Feature selection and encoding
- Handling imbalance 
- Model training and evaluation
- Visualizations: Heatmaps

## Requirements
- Python 3.8+
- pandas, numpy
- scikit-learn
- imbalanced-learn
- matplotlib, seaborn
